# tgdrive

Файловый навигатор для Telegram. Бот предоставляет доступ к файловой системе сервера прямо из мессенджера — просмотр директорий, скачивание файлов и выполнение shell-команд.

## Возможности

- **Навигация по дискy** — директории и файлы отображаются в виде inline-кнопок с пагинацией
- **Отправка файлов** — при нажатии на файл бот отправляет его в чат (до 50 МБ)
- **Shell-команды** — любое текстовое сообщение выполняется как команда в текущей директории
- **Dot-файлы** — кнопка 👁 включает/скрывает скрытые файлы
- **Команды** — `/start`, `/pwd`, `/ls`, `/cd <path>`
- **Авторизация** — доступ только для указанных Telegram ID
- **Прокси** — поддержка HTTP/SOCKS5 прокси

## Требования

- PHP ≥ 8.0 (CLI)
- Расширения: `curl`, `json`, `mbstring`, `posix`

## Установка

```bash
# Установить скрипт
sudo cp tgdrive.php /usr/bin/tgdrive
sudo chmod +x /usr/bin/tgdrive

# Запустить — создаст конфиг и завершится
tgdrive

# Заполнить конфигурацию
nano ~/.config/tgdrive/config.php
```

Конфигурация выглядит так:

```php
<?php
$tg_token = '123456:ABC-DEF...';
$proxy_url = '';            // или 'socks5://127.0.0.1:9050'
$user_ids = [123456789];    // Telegram ID допущенных пользователей
```

После заполнения запустите бота повторно:

```bash
tgdrive
```

## Systemd (автозапуск)

Установка юнита для текущего пользователя:

```bash
mkdir -p ~/.config/systemd/user
cp tgdrive.service ~/.config/systemd/user/
systemctl --user daemon-reload
systemctl --user enable --now tgdrive
```

Логи:

```bash
journalctl --user -u tgdrive -f
```

## Использование

### Навигация

Бот отображает содержимое директории в виде inline-кнопок. Папки отмечены 📁, файлы — 📄. В нижнем ряду:

| Кнопка | Действие |
|---|---|
| ⬆️ Вверх | Перейти в родительскую директорию |
| 🏠 Дом | Вернуться в домашнюю директорию |
| 👁 | Показать/скрыть файлы начинающиеся с точки |
| ◀️ ▶️ | Листать страницы, если файлов много |

Нажатие на папку — вход в неё. Нажатие на файл — отправка в чат.

### Команды бота

| Команда | Описание |
|---|---|
| `/start` | Сбросить состояние, начать с домашней директории |
| `/pwd` | Показать текущую директорию |
| `/ls` | Полный список файлов текстом |
| `/cd <path>` | Перейти в указанную директорию (абсолютный или относительный путь, `~`, `..`) |

### Shell-команды

Любое текстовое сообщение, не начинающееся с `/`, выполняется как shell-команда в текущей директории. Вывод (stdout + stderr) отправляется обратно в чат.
